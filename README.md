uncompile_pyc
=================================== 
安装：
-----------------------------------
### 方法1 从安装包安装
'pip install decompile_pyc'
### 方法2 源码安装
'git clone https://github.com/Garen-in-bush/Decomplie_pyc.git'
进入到setup.py所在的目录，执行
'python setup.py sdist bdist_wheel'
'cd dist'
'pip install xxx.tar.gz'
或者
‘pip install xxx.wheel’
